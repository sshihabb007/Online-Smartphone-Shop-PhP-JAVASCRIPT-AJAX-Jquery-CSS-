A full-stack, dynamic e-commerce web application designed specifically for browsing and purchasing smartphones. Built using a robust combination of **PHP, JavaScript, AJAX, jQuery, HTML, and CSS**, this project features a user-friendly storefront for customers and a secure, dedicated Admin Panel for managing inventory and orders.

## 🛠️ Technology Stack

This project is built purely with native web technologies, without heavy reliance on modern reactive frameworks, demonstrating a strong understanding of core web development concepts.

* **Frontend:** HTML5, CSS3, JavaScript (ES6+), jQuery
* **Backend:** PHP
* **Database:** MySQL
* **Asynchronous Operations:** AJAX (for seamless data fetching and UI updates without page reloads)

---

## ✨ Key Features

### For Users / Customers
* **Interactive Storefront:** A clean, responsive user interface to browse various smartphone brands and models.
* **Dynamic Search & Filtering:** Powered by AJAX and jQuery, allowing users to search for specific phones or filter by categories instantly without refreshing the page.
* **Product Details:** Detailed view for each smartphone, displaying specifications, price, and high-quality images.
* **Shopping Cart System:** Users can add items to their cart and proceed to checkout smoothly. 

### For Administrators (Admin Panel)
* **Secure Authentication:** Protected login system restricting access to authorized personnel only.
* **Inventory Management:** Add, update, or delete smartphone listings, including uploading images and modifying prices/specs.
* **Order Tracking:** View customer orders and manage their fulfillment status.
* **User Management:** Oversee registered customers and handle user data.
